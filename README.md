# SIMPLE REPORT, S.L. — SPA (React + Tailwind + Vite)

## Requisitos
- Node.js 18+
- npm

## Instalación
```bash
npm install
npm run dev
```

La app se abrirá en `http://localhost:5173`.

## Construir producción
```bash
npm run build
npm run preview
```

## Notas
- Rutas principales en `src/App.jsx`.
- Tailwind configurado en `tailwind.config.js` y `src/index.css`.
- Imágenes de ejemplo en `/public` (cámbialas por las definitivas).
